Projeto criado para a disciplina de Humanidades e Cidadania do terceiro semestre do curso de engenharias da faculdade UnB - FGA. Como pedido, ele demostra o uso de uma tecnologia estudada no trabalho final, 
o projeto consiste em um avião que pode ser controlado em um cenário de duas dimensões, para isso o usuário pode optar por controlar o avião utilizando as teclas de setas do teclado 
(up arrow = gira o avião em sentido anti-horário, down arrow = gira o avião em sentido horário, right arrow = acelera o avião e muda a direção para ficar apontado para a direita, tecla de espaço = aumenta a aceleração do avião, "p" = pausa a simulação, "a" = ativa o piloto automâtico). Além disso, o piloto automático também é ativado sozinho caso o avião não seja controlado por um período de 5 segundos.
